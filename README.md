# SynergyX | R&D Hub 🚀

**Built at HackBMU 8.0 by Team SynergyX**

SynergyX is a centralized Research & Development (R&D) directory and collaboration platform. It bridges the gap between students and top faculty, making it easier to discover active research projects, connect with mentors, and drive global innovation.

## 🌟 Key Features
* **Role-Based Authentication:** Dedicated and secure portals for both Students and Faculty.
* **Faculty Directory:** Explore profiles of top faculty members and their active R&D projects.
* **Seamless Connectivity:** Streamlined process to save research hours and find the right mentorship.
* **AI Assistant Integration:** Quick query resolution for navigating projects and faculty details.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT / Bcrypt (Secure Login/Signup)
